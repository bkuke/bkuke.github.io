你的需求很特别，你要最符合你心意的CDDA，拒绝大众化。 那就全特性安装，满足你的一切要求。当然，有技术门槛。

很显然,你是资深用户,熟练使用git,科学代理,linux,编译个程序那都不算个事,那我就不多BB了：

  1. github官网
  2. 阅读编译文档
  3. 示例代码如下：
    $git config http.proxy='socks5://127.0.0.1:1080'
    //没有代理下载2个多G,你会疯的
    $git config pull.rebase=true

    $cd Cataclysm-DDA
    //进入git文件夹
    $git log --oneline
    //通过日志来查看版本数据以及对应的哈希码
    $git checkout xxxx
    //这个xxxx是版本哈希数据，从而自由切换版本。
    //你可以选择最稳定的版本来编译。如果你高兴，甚至可以玩0.1版本的CDDA。
    //至于哪个是最稳定的版本，查看tag来判断
    //个人推荐最新版本，那就是$git checkout master
    //以下仅以我个人的linux（64位）sdl编译为例：
    $sudo apt-get install libsdl2-dev libsdl2-ttf-dev libsdl2-image-dev libsdl2-mixer-dev libfreetype6-dev build-essential astyle
    //编译环境,依赖项;make astyle可以用归整代码（前提是你魔改过源代码）,不然无法重新make
    $make -j4 TILES=1 SOUND=1 RELEASE=1 USE_HOME_DIR=1

    $make localization zh-CN
    //默认编译语言没有中文,添加中文翻译文件
    $./cataclysm-launcher
    //启动！

[属性]全降到最低(4),因为模组里有[属性跟随技能值提高],会加强你设置的属性。

这个[属性跟随技能]mod,有一个极其复杂的公式,根据某个属性对应6或者7个技能等级的总和加加减减再取0.4次幂,所以一般来讲,最好人物素质全面发展（当然,首先你要活着,然后才能发展）,不明白就画个图y=x^0.4,x轴是技能等级的和,y轴是获得的属性点,素质教育才能快速提升属性。就拿力量属性来讲,6个对应的技能都到10级,这个属性差不多能加5点属性,然后就难了。就算6个技能都20级（关于等级，见下文）了，也就多加1个技能点。
不BB,上源码（注意：源码也是会变的,具体实现视版本而定）:

"skill_offset": -3, //等级和加个偏移量-3
"scaling_power": 0.4 //偏移量计算之后的结果再0.4次方得到最终结果

避难所

好了，进入正题，学会：挨饿+挨打+捡垃圾，素质三连，就能横行村里，成为一方霸主了。

首先，不知道怎么才能挖地道，施法，开车，使用电脑，看任务，看地图。。。blabla，记得按英文问号键，查看按键设置。 出现在避难所，图像太小按z键扩大，shift+z缩小。图片右下角是一台电脑，黑暗中发光看到没。
有npc可以交谈(C键)，你当然可以不鸟她，因为新手活不长定律，她给你的任务怕是完不成的。

如图，c键接任务，d键[组队]可能不鸟你，也可能同意（如果你长得帅。就是人物特性设置的，比如长个尾巴，不像人，大多数npc会不喜欢你，少数npc会爱‘上’你），e是npc带的物品交易（杀了她也能获得），g就是结束对话了（其他请自己摸索，这是游戏乐趣）。下方的审视，估测，观察，都是看npc状态以及对你的态度。也可以无视npc，毕竟有的闸总npc队友是会半夜偷袭你，或者偷你的东西，然后跑路，让你一觉睡醒回到解放前。建议新手暂时不理她，专注于游戏其他内容，毕竟这个npc是个静态npc，跑不了。

说起来，长得丑也不用担心，变异可以拯救你，[精灵变异]，倾国倾城那都不是事。总之，人丑就要多读书，书里有变异配方，至于配方究竟在哪，变异有哪些，怎么变，我们在第七节再说。

来到电脑旁边（不会走路？给你一个英文问号，自己体会），e键问你检查哪里，按方向键指向电脑，显示上方的图片，3键[联系我们]接受任务，大地图自动生成路线。可能会卡一下，然后按M键打开任务，如图,第一个任务绿色，表示大地图追踪的是难民中心的位置。第二个是白色，表示没有追踪。


按@，显示个人状态，就拿智力来说，创造人物时设置为4点，[技能增加属性]mod额外增加的属性为2点，现在是6点。总之，括号里面的数字（基础属性）一般不用管，左边的是当前属性（包括吃药获得的临时属性）。
现在我选择（方向键谢谢）敏捷，按下回车，屏幕中间提示：点数不足。下一级经验：3

因为有杀怪加点mod，可以加属性点，提示你经验值不够，经验值超过3点才会升到1级，获得1个自由属性点，为什么你的不是3，那是因为我魔改了源代码(是的，这个mod写进的cpp里，不是改改json就行的。 )，在src/avatar.cpp里面，原本是300点，现在改成了3点，如果你也要改（我认为新手可以适当强化乃至神化自己的人物，以渡过前期，不至于3分钟弃坑），改完记得make astyle，然后重新make编译。

然后，状态栏左下方，技能栏，选择（就是把蓝色的游标移到上边）一个技能，按回车，就能取消练习状态，什么意思呢，就是平时做饭消耗很多体力和精力，精力就是用于学习了，学习就会涨经验值。取消练习状态，让你节省精力，不那么快疲劳，但是技能不涨经验，不升级。不建议新手点，之所以介绍，是防止你误点，然后升不了级，然后弃坑。
在上图中，烹调和生存就被取消练习了（深蓝色），制造技能是5级，0%是升级进度，99%之后再练习一次，就到6级了，然后进度又回到0%。如此反复。

吃顿饱的

怎么吃饱呢？当然是探索城镇，看到吃的你就拿，千万别客气。还有记得睡觉。 什么薯条，汽水，面团，饼干，不讲究，能吃饱就行。

上面的看起来简单，但其实有难度。探索城镇，就包括冒雨出行（甚至酸雨），丧尸巷战，遭遇战，伏击战。在有弩（又被削了。。心累）的情况下，远程风筝两三个[普通]丧尸不是大问题。打完记得把没坏的箭收回来啊。

看到吃的就拿，这个也有难度，带不下了怎么办，[生化信徒]的背包算是挺好的背包了。但还是带不下。尤其是你需要带打火机，手机，火盆，汽水罐，搜集的工具，这些工具是为未来做准备的，你也不想捡一辈子垃圾--吧（文明你我他，谢谢）。

吃顿好的

你已经吃了很多快餐，垃圾食品，精通全地形作战，绝地反杀不是个事，但是你不想再这样下去了，你要吃顿好的！

你有瑞士军手，但是最好还是用打火机，省资源。你还需要一个锅，铜锅，铁锅，能做饭就是好锅，好锅：

具有烹调，煮沸，容器这些属性就行，名字不重要，它就是叫脸盆，只要有这些属性，那就是锅。
然后是火盆（a键激活，选择放置位置，然后D键把木材扔进去），火盆里面放东西（能烧的）就可以烧火，它的特点就是不会烧着房子，只在火盆那一格着火。不论房子是什么地板，你都不应该在屋里放火，房子一般会被烧塌。野外纵火也有风险，会烧毁整片森林（海量资源都没了），所谓放火烧山，牢底坐穿。
那么这么好的东西去哪买呢？在出生点，[拆解]铁柜是有[薄钢板]的，注意，是拆解。其次，是冰箱拆解/砸碎都可以获得薄钢板。这是最常见的来源。有了薄钢板，就可以制造火盆了。
最后，有了火，有了锅，就可以烧水喝+加热食物（行为快捷键%）+做饭（如果你有原料的话）。
